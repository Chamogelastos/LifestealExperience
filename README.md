**A modpack to simulate the lifesteal experience**

Use the lifesteal mod to manage the heart system and revive ability

Have Simple Voice Chat to make everything more immersive

And make your server like you're playing on the real ONE! So have a good time

# Setting up the mod
Everything is happening in the config folder, so go and open it up. In the lifesteal-common.toml file is the lifesteal config file. There the basics you can change these things:

-Starting Hearts

-How many hearts you lose on death, or how many you get on a kill

-If you drop a revive head on complete elimination

-The Complete Dead people if they get banned or are in spectator mode

-How many seconds to auto revive

-When to lose hearts (If mobsm environment or people

To set up the Voice Chat go to the config file and change the port to the one your server has

**And now you are done**
